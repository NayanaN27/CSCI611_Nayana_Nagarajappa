# Assignment 4 - Neural Style Transfer

## Description
This project implements Neural Style Transfer using a pre-trained VGG19 model.

## Files
- style_transfer_exercise.ipynb → main implementation
- Report_Assignment_4.pdf → report with experiments and analysis
- images/ → content and style images

## How to Run
1. Open the notebook in Jupyter or Google Colab
2. Upload content and style images
3. Run all cells sequentially
4. The final stylized image will be displayed

## Experiments
Different hyperparameters (content_weight, style_weight) were tested and results are included in the report.
